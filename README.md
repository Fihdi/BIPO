# 8hp Dual Bipolar-VCAs / Ring modulators

![](https://raw.githubusercontent.com/Fihdi/BIPO/refs/heads/main/BIPO.png)

BIPO consists of two VCAs which can be used individually or locked together to allow for sidechaining / crossfading. Changing the LEVEL, the VCAs can be biased as to allow each section to be used as a ringmodulator. The resulting ringmodulated signal of the input and the CV is available at the BIPO outputs. The VCA outputs are always summed together at the MIX output.
